富邦金(2881) 指標類技術分析 2023.05.20

2023-05-20
2023-05-21
股票技術分析

由於適逢清明連假，股市休市至4月6日，本次技術分析實際使用的資料區間為2023年4月6日至5月2日，共計18個交易日。

A. 價格指標

1. KD 值

KD值又稱為隨機指標(Stochastic Oscillator)，是由K值和D值所組成的兩條線圖，我們在股市技術常聽到的死亡交叉和黃金交叉指的就是KD指標。不論K值或D值反映的都是股價的變化速度，兩者的數值皆落於0~100之間。

K值 : 快速平均值，對股價變化的反應較靈敏、快速。
D值 : 慢速平均值，對股價變化的反應較不靈敏。

  • K值 > D值，代表有上漲行情，適合做多(黃金交叉，買進訊號)。
  • K值 < D值，代表有下跌行情，適合做空(死亡交叉，賣出訊號)。
  • KD值大於80，表示目前處於超買區。
  • KD值小於20，表示目前處於超賣區。

補充說明: 多數股市資訊網站的KD值，都是以K9和D9為主，表示以過去9天的股價進行計算。

由下方的KD值圖表可觀察到自4月10日至4月19日的8個交易日，K值均大於80處於超買區。在4月19日出現死亡交叉(K值 < D值)，為賣出訊號，富邦金的股價呈現下跌趨勢。而在4月28日則出現黃金交叉(K值 > D值)，為買入訊號，富邦金的股價呈現上漲趨勢。


(圖表來源: Goodinfo!台灣股市資訊網)

2. RSI

RSI又稱為相對強弱指標(Relative Strength Index)、逆勢指標，是以過去某段時間股價的平均漲幅和平均跌幅計算出的數值，可以用來衡量買盤、賣盤雙方力道的強弱，以及觀察市場是否過熱或過冷。RSI值的範圍在0~100之間，數值越大代表過去漲的機率越大，假設這個趨勢不變，未來有可能也會漲的機率比較大。

  • RSI值 > 70 (80)，為買超訊號，代表市場過熱，股價可能下跌。
  • RSI值 < 30 (20)，為賣超訊號，代表市場過冷，股價可能上漲。

若以RSI值大於70和小於30作為判斷標準，則由下方的RSI值圖表可觀察到自4月12日至4月18日的5個交易日，RSI值均大於70處於超買區。在4月19日RSI值下降至64.05脫離超買區並剛好處於死亡交叉(K值 < D值)，富邦金的股價開始出現下跌趨勢。往後自4月19日到5月2日的9個交易日，RSI值均處於低檔，但均沒有落在超賣區(RSI值 < 30)，期間最低的RSI值(36.44)出現在4月25日，隔日富邦金股價則有些微反彈。


(圖表來源: Goodinfo!台灣股市資訊網)

補充說明: 上面的RSI值指的是RSI5，也就是以過去5日股價的平均漲幅和平均跌幅計算出的數值。

3. MACD

MACD又稱為指數平滑異同移動平均線(Moving Average Convergence)，透過計算「收盤時股價或指數變化的指數移動平均值(EMA)」之間的離差程度(DIF)而來，用來確定波段漲幅並找到買賣點，MACD指標由DIF和MACD兩條線所組成。

  • EMA(指數平滑移動平均線)
    • 最近的股價配以較高權重，越遠的股價配以較低權重，以不同股價權重計算出的數值。
    • 可以避免單日股價漲幅過大，導致移動平均的數值被大幅影響。
  • DIF(離差值)
    • 可以判斷股價短期的趨勢變化。
    • 利用短期(12日)與長期(26日)的指數移動平均相減計算出的數值。
    • DIF = EMA(12) - EMA(26)。
  • MACD
    • 可以判斷股價長期的趨勢變化。
    • 利用DIF再取一次移動平均而計算出的數值。(一般是取DIF的9日移動平均)
    • MACD = EMA(DIF,9)。
  • OSC
    • 可以判斷股價趨勢反轉的買賣點。
    • 利用MACD和DIF計算出的數值。
    • OSC = DIF - MACD。
  • DIF與MACD都在0軸線上，表示目前為牛市。(多頭市場)
  • DIF與MACD都在0軸線下，表示目前為熊市。(空頭市場)
  • DIF值 > MACD值，並且DIF值 > 0，為買進訊號。
  • DIF值 < MACD值，並且DIF值 < 0，為賣出訊號。
  • OSC由負轉正，為短線買進訊號。
  • OSC由正轉負，為短線賣出訊號。

由下方的MACD圖表可以觀察到自4月6日至5月2日的18個交易日，OSC值(柱狀圖)均大於0，股價趨勢並沒有發生轉變。DIF在18個交易日內，始終都大於MACD，而在4月18日DIF才突破0軸線，並出現買進訊號，但是仔細看圖表就會發現目前富邦金的股價處於相對高點的位置(58.6)，在DIF剛突破0軸線就買入，就等於直接買在高點，所以參考MACD指標可能對富邦金這檔股票的短線評估不是那麼有效。


(圖表來源: Goodinfo!台灣股市資訊網)

4. 乖離率(Bias)

乖離率(Bias Ratio)是股價和均線的距離，而就趨勢而言，股價會往均線方向移動。我們可以經由乖離率來衡量目前股價偏離平均移動線的程度，並評估股票是不是漲太多或跌太深。除此之外，我們也能視乖離率為某個時期的平均報酬率。

  • 乖離率(Bias Ratio)
    • 可以評估股價偏離移動平均線的程度要多大，股價會再度向移動平均線靠近。
    • 乖離率 = (目前價 - 移動平均價) ÷ 移動平均價。
    • 股價等於移動平均線，乖離率為0。
    • 股價大於移動平均線，乖離率為正，反之則為負。
  • 5日乖離率達到3.5%(-3.5%)，代表有超買(超賣)情形。
  • 10日乖離率達到5%(-5%)，代表有超買(超賣)情形。
  • 20日乖離率達到8%(-8%)，代表有超買(超賣)情形。

(補充說明: 乖離率標準因人而異，僅供參考。)

因為我們使用的是最近1個月的資料，來評估5月3日和4日的股價表現，所以在下方的乖離率圖表中，我們參考的是月線(20日線)，也因此適用20日乖離率的超買、超賣百分比標準。在4月6日到5月2日的18個交易日中，20日乖離率在4月14日達到最高的3.65%，而綜觀18個交易日均並未達到正負8%的20日乖離率標準，從乖離率看不出任何關於富邦金個股的超買和超賣訊號。


(圖表來源: Goodinfo!台灣股市資訊網)

5. DMI

DMI又稱為趨向指標(Directional Movement Index)，用於中長期判斷多頭和空頭勢力，不適用於短期操作。DMI圖表由+DI、-DI和ADX組成，其中+DI和-DI是用來判斷趨勢的方向，而ADX則是用於衡量趨勢的強度。(DMI數值計算有點複雜暫且跳過。)

  • +DI由下向上突破-DI，表示買進訊號。
  • -DI由下向上突破+DI，表示賣出訊號。
  • ADX逐漸增加，代表市場往上漲方向移動。
  • ADX從上升轉為下跌，代表行情即將反轉。(股價下跌)
  • ADX從下跌轉為上升，代表市場即將反彈。(股價上漲)

由下方的DMI圖表可以觀察到在4月13日，+DI由下向上突破-DI，表示買進訊號，此時富邦金的股價確實處於相對低點。而後續到5月2日的所有交易日，+DI均在-DI之上，這時我們將關注點轉向ADX，可以發現在4月19日到4月27日的7個交易日，ADX從上升轉為緩慢下降的趨勢，同時富邦金股價亦呈下跌趨勢。但在4月28日，ADX又從下跌轉為上升，代表市場即將反彈，並且在4月28日富邦金的股價確實由每股57.7元反彈至每股59元。


(圖表來源: Goodinfo!台灣股市資訊網)

B. 成交量指標

1. 平均成交量

成交量指的是在某一段時間內具體的交易數量，成交量變化可以反應資金進入市場的狀況。一般而言，在講到成交量時，會包含2個部分，分別是成交股數和成交金額，但不同標的股價高低不同，同樣的成交股數，最後的成交金額差異甚鉅，所以實務上講的成交量多半是以成交金額為主。

  • 量是價的先行指標，價量背離是變盤的前兆。
  • 成交量擴大是股價上漲的必要條件。
    • 成交量小有上漲力道不足、股票難脫手等問題。
    • 成交量大代表投資人熱衷股市，應做好出場準備。(賣出股票)
    • 成交量小代表投資人看壞股市，應做好進場準備。(買進股票)
  • 價量關係
    • 價漲量增 : 價格上漲，成交量增加。
    • 價跌量縮 : 價格下跌，成交量減少。
  • 價量背離
    • 價漲量跌 : 股價上漲，成交量減少，若股價已大幅上升，則代表後續推升力道不足。
    • 價跌量漲 : 股價下跌，成交量增加，代表有更多買盤的交易量接手，有時是止跌反轉訊號。
  • 股價的高峰(谷底)往往出現在交易量大(小)的時候。

由下方的成交量圖表可以觀察到在4月28日是富邦金個股成交量最大的交易日，同日股價也達到高峰。而從4月7日至4月14日的6個交易日，交易量呈現上升趨勢，帶動股價持續上漲。後續4月17日至4月25日，每日交易量都相較4月14日萎縮許多，連帶股價持續下跌，但在4月26日交易量卻上升許多，表示股價有反轉上漲的機會，並在4月28日得到證實，當日交易量衝高到1.83萬張，股價上漲到區間的最高價。


(圖表來源: Goodinfo!台灣股市資訊網)

2. OBV

OBV又稱為能量潮(On Balance Volume)，也有人稱其為人氣指標，是一種依據行情的漲跌，來累計或刪去市場的成交量值，而以此累算值作為市場行情動能變化趨勢的指標。OBV由葛蘭碧提出，她認為市場的動能應該是反應在成交量的變化，並且量是價的先行指標。(量先價行)

  • OBV
    • 可以代表個股在市場上人氣的興衰。
    • 先給定初始的OBV，之後依據股價漲跌來累計市每日成交量。
    • 如果t日收盤價 > t-1日收盤價，OBV(t) = OBV(t-1) + 本日交易量。
    • 如果t日收盤價 < t-1日收盤價，OBV(t) = OBV(t-1) - 本日交易量。
  • OBV線上升，股價趨勢線下跌，表示買進訊號。
  • OBV線下降，股價趨勢線上升，表示賣出訊號。
  • OBV線從負的累積數轉為正數，表示買進訊號。
  • OBV線從正的累積數轉為負數，表示賣出訊號。
  • OBV線緩慢上升，表示買進訊號。
  • OBV線暴漲、暴跌，表示無法判斷。

由下方的OBV圖表可以觀察到自4月6日到4月14日的7個交易日，OBV線整體呈現上升趨勢，同時富邦金的股價月線正在緩慢下跌，表示買進訊號，後續富邦金的股價確實持續上漲。但自4月17日到4月25日的7個交易日，OBV線整體呈現向下趨勢，同時富邦金的股價月線處於緩慢上升的趨勢，表示賣出訊號，後續富邦金的股價確實緩慢走跌。


(圖表來源: Goodinfo!台灣股市資訊網)
