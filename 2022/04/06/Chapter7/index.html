CSS Flex Box Chapter7.

2022-04-06
2022-04-06
CSS Flex Box, 網頁基礎
material from: freeCodeCamp、CSS Tools: Reset CSS

Final Summary

現在寫網頁時，對於使用者體驗這塊非常注重。以電商網站為例，顧客在網頁上停留的越久，對於店家商品的銷售越有利，所以要怎麼讓使用者獲得比較好的網頁瀏覽體驗呢?

其中一種比較簡單的方式就是 RWD (Responsive Web Design) 也就是互動式網頁設計，讓網頁能適應使用者所用的瀏覽裝置螢幕尺寸，做出相應版面的調整。

不做 RWD 的話，或許網頁在電腦或螢幕尺寸較大的裝置上會有不錯的瀏覽體驗，但到了平板或手機等螢幕較小的裝置，瀏覽體驗可能就不是那麼好，像是字太小、版面雜亂等等。

而 Flex Box 其實就是 RWD 當中的基礎，在看完 flex box 所有的章節後，想必你對有無 RWD 的差別會有一定的認識吧! 在 RWD 的領域裡，其實還有更多、更艱深的技術等著被探索，如果有興趣不妨深入研究一番，想必你會有滿滿的收穫~

最後，就用一張圖結束這回合吧~ (As a Japanese ACGN Culture Lover…)
