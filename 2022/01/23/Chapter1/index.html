Use display: flex to Position Two Boxes

這邊算是 flex box 的初入門，display:flex 宣告父容器要用 flex box 的形式放置網頁元素。

因為下面設定2個box各佔50%寬度的關係而且沒有多寫flex: wrap;，所以在縮放視窗時，沒辦法感受到 flex box 的威力，原則上所有子元素都會被排在同一列上。
